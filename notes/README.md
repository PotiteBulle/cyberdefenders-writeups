# Notes

This folder contains general notes used across CyberDefenders labs.

These notes are not tied to a single lab. They are meant to support my methodology, tooling, command usage and SOC vocabulary.

## Files

| File | Description |
|---|---|
| `methodology.md` | General investigation methodology |
| `tools.md` | Tools used for SOC, DFIR, malware triage and threat intelligence |
| `commands-cheatsheet.md` | Useful commands for labs |
| `soc-glossary.md` | SOC and Blue Team glossary |

## Purpose

The goal of this folder is to keep reusable knowledge in one place.

It helps maintain consistency between writeups and provides quick references during investigations.

## Usage

These notes can be updated over time as I complete more CyberDefenders labs.

They are written in English to keep the repository accessible and professional.
