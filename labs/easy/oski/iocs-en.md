## 11. Observed Cleanup Command

The following command was observed in the child `cmd.exe` process.

```cmd
cmd.exe /c timeout /t * & del /f /q "C:\Users\admin\AppData\Local\Temp\VPN.exe" & del "C:\***********\*.dll" & exit
```

This command shows:

- a delay before deletion
- deletion of the temporary binary
- deletion of DLL files in a specific directory
- post-execution cleanup logic

## 15. Usage Notes

These IOCs should be used in an educational and defensive context.

For a real investigation, it would be necessary to:

- check whether the IOCs are still active
- enrich IPs and domains with additional sources
- correlate with proxy, DNS, EDR and SIEM logs
- check for the presence of hashes on endpoints
- hunt for cleanup commands in process logs
- create detection rules when appropriate
